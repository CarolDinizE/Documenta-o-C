Introdução à Lógica e Conceito de Algoritmo

A **Introdução à Lógica de Programação** fundamenta todo o processo de desenvolvimento de software. Ela consiste no uso correto das leis do pensamento e do raciocínio humano aplicadas à estruturação de instruções que um sistema de computação consiga compreender de maneira lógica e sequencial. Dominar esse conceito permite que o programador organize suas ideias de forma clara antes mesmo de escrever as primeiras linhas de código em qualquer linguagem.

Intimamente ligado à lógica, temos o **Conceito de Algoritmo**. Um algoritmo funciona exatamente como uma receita de bolo bem detalhada: ele representa uma sequência finita de passos lógicos, ordenados e sem ambiguidades que têm como único objetivo resolver um problema ou executar uma tarefa específica. Na ciência da computação, os algoritmos servem de rascunho para a lógica que será implementada no computador.

No desenvolvimento de sistemas usando a linguagem C, criamos programas combinando essas regras formais de lógica com blocos estruturados de código, transformando as instruções lógicas em softwares funcionais de alto desempenho.

Programação Estruturada

A **Programação estruturada** constitui um paradigma de desenvolvimento criado para organizar e simplificar a escrita de softwares. Esse modelo baseia-se fortemente na quebra de problemas complexos em pequenos blocos de comandos isolados. Ele foca o desenvolvimento em apenas três pilares básicos de controle: estruturas de sequência linear, estruturas de decisão (condições) e estruturas de iteração (loops).

Ao contrário dos estilos antigos de desenvolvimento que usavam saltos lógicos desordenados (como o temido comando GOTO), a programação estruturada incentiva a criação de rotinas limpas, facilitando a manutenção e a leitura do código por outros desenvolvedores. Na linguagem C, implementamos esse conceito criando funções modulares que conversam entre si.

Qualquer programa clássico escrito em C precisa, obrigatoriamente, possuir uma função inicial bem definida. É a partir dessa estrutura que o sistema operacional sabe onde a execução deve começar:

int main() { // As instruções do seu algoritmo ficam aqui dentro return 0; }
Constantes, Variáveis e Tipos de Dados

**Constantes e variáveis** são elementos que representam espaços reservados diretamente na memória RAM do computador para fins de armazenamento temporário de dados. Uma variável é um identificador cujo valor salvo pode sofrer alterações dinâmicas conforme as instruções do programa rodam. Por outro lado, as constantes armazenam um valor imutável, definido no início do código, que o sistema não pode alterar em nenhuma hipótese.

Para criar essas caixas de armazenamento, a linguagem exige a definição dos **Tipos de dados** correspondentes. O tipo avisa ao sistema operacional quanta memória RAM deve ser separada para aquela informação. Veja a lista com os tipos primitivos fundamentais disponíveis nativamente em C:

  • int: Tipo de dado específico para o armazenamento de números inteiros positivos ou inteiros negativos.
  • float: Utilizado para armazenar números reais com casas decimais (ponto flutuante com precisão simples).
  • double: Destinado a números reais decimais extensos que exigem uma precisão muito maior de armazenamento.
  • char: O menor tipo de dado nativo, reservado para guardar um único caractere textual ou símbolo da tabela ASCII.

O processo de criação de uma variável obedece a uma sintaxe estrita: definimos primeiro o tipo e depois damos um nome único para o identificador. Confira no exemplo prático:

int idade = 19; float altura = 1.65; const double PI = 3.14159;
Entrada, Saída e Operadores Aritméticos

As rotinas de **Entrada e saída** de dados permitem que o nosso programa interaja diretamente com o mundo externo, recebendo comandos através do usuário ou exibindo resultados processados de volta na tela. Como a linguagem C não possui comandos nativos de I/O em seu núcleo, recorremos à biblioteca padrão stdio.h para disponibilizar essas ferramentas essenciais.

Além da manipulação das informações de entrada e saída, usamos constantemente os **Operadores aritméticos** no código. Eles realizam as operações matemáticas tradicionais com as variáveis e valores numéricos do sistema. A lista abaixo apresenta os principais comandos e símbolos de cálculo usados no dia a dia:

  • printf(): Função responsável pela saída de dados, exibindo mensagens e resultados formatados na tela do monitor.
  • scanf(): Função de entrada, responsável por pausar a execução do sistema para ler o que o usuário digita no teclado.
  • + e -: Operadores matemáticos padrão para efetuar somas e subtrações matemáticas básicas.
  • * e /: Operadores matemáticos utilizados para efetuar multiplicações e divisões dentro do código.
  • %: Operador aritmético especial responsável por calcular o resto exato de uma divisão entre dois números inteiros.

O exemplo abaixo demonstra como capturar um número inteiro digitado pelo teclado do usuário e exibir o valor logo em seguida:

printf("Digite o seu ano de nascimento: "); scanf("%d", &ano); printf("Você digitou o ano %d", ano);
Estruturas Condicionais, Relacionais e Lógicas

As **Estruturas condicionais** mudam o comportamento do programa criando bifurcações no fluxo de processamento. Em vez do código rodar apenas de cima para baixo de forma engessada, as condicionais forçam o computador a tomar decisões com base no resultado verdadeiro ou falso de um teste lógico pré-determinado.

Para construir essas validações, aplicamos os **Operadores relacionais**. Eles fazem testes matemáticos de comparação direta entre dois valores ou variáveis, devolvendo um resultado lógico. O C usa símbolos como == para checar igualdades, != para diferenças, além de > (maior) e < (menor).

Quando a tomada de decisão envolve mais de uma regra, entram em ação os **Operadores lógicos**. Eles servem para fundir ou inverter múltiplos testes relacionais. Usamos o operador && (E lógico) que exige que todas as condições sejam verdadeiras, o operador || (OU lógico) que exige apenas uma condição verdadeira, e o operador ! (NÃO lógico) que inverte o estado lógico atual.

if (idade >= 18 && temCarteira == 1) { printf("Acesso autorizado!"); } else { printf("Acesso negado."); }
